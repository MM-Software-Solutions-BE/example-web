# example-web

Starter: **Next.js 16** + **Tailwind v4** + **shadcn** (base-nova), zelfde basis als `roots-connect-web` / `atlas-coding-web`.

- Zelfde **22 bestanden** in git als die eerste commits (o.a. `AGENTS.md`, `CLAUDE.md`, `components.json`, default `page.tsx`). `next-env.d.ts` staat in `.gitignore` en wordt lokaal gegenereerd.
- Extra runtime-deps t.o.v. die eerste push: `cmdk`, `sonner`, `lucide-react@^1.6` — handig om meteen verder te bouwen zoals Atlas.

```bash
npm run dev
npm run build
```
